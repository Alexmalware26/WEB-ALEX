# HadesCreative

Landing page de servicios freelance de diseño y desarrollo web.

**Live:** desplegado en Vercel desde la rama `main`.

## Estructura

```
/
├── index.html         Página principal
├── faq.html           Preguntas frecuentes
├── proyectos.html     Clientes y casos
├── robots.txt         SEO
├── sitemap.xml        SEO
├── css/
│   ├── global.css     Variables, base, splash, scroll, back-to-top
│   ├── navbar.css     Barra de navegación
│   ├── hero.css       Sección hero
│   └── sections.css   Resto de secciones
├── js/
│   ├── smoke-background.js  Shader WebGL de fondo
│   ├── effects.js     Contador animado de precios
│   ├── ui.js          Formulario, menú móvil, comparador
│   └── ux.js          Splash, scroll, back-to-top
└── assets/
    ├── favicon.svg
    ├── og-image.svg
    └── images/        Capturas de proyectos
```

## Stack

HTML + CSS + Vanilla JS — sin frameworks ni build step. Form delivery via Formspree.
